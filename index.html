Overview

Windows developers and users sometimes face a necessity to debug or to adjust their or third party executables or other PE files. A various range of tools and software are available for those purposes nowadays. Some tools cover specific areas of lowlevel operations, other ones are complex and powerful solutions attempting to provide as much features as possible. Binripper delivers a compromise between singlepurpose tools simplicity and utility packages selfsufficiency.

Biripper is a tool for advanced Windows users and developers who inspect and tune Windows applications, libraries, and installers in their everyday work. It consists of a set of plugins each one handling specific lowlevel area features combined with one simple user interface.

Binripper has been designed by people with over a decade's experience in system programming so it includes the features that appeared to be essential in routine work of system developers.

Plugins

The tool includes the following plugins:

Disassembler with handy multiview disassembly listing and providing users with ability to mix binary code and to increase code section size

NSIS Viewer allowing users to browse files packed in NSISbased installers and to extract or to replace them

PE Info that shows PE file details (header, certificate, data directories, sections, relocations, import/export tables) and allows of recalculating section headers and removing certificate and relocations.

Biripper roadmap includes further improvement of its functionality coming soon (Resource Editor, HEX Editor, 64bit Disassembler)
